Bienvenido

Entra con tu usuario o crea una cuenta. El administrador recibe la notificación del registro sin exponer contraseñas por correo.

El sistema guarda las contraseñas cifradas en la base de datos. El correo del administrador recibe nombre, email, usuario y fecha del registro, no la contraseña.
Si abres solo el HTML con doble clic, el login no funcionará. Esta versión debe ejecutarse con el servidor incluido para que la base de datos y el envío de correo funcionen correctamente.
